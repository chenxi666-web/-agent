# -agent
我构建了一个基于多 Agent 架构的个人知识管理系统，用于解决知识碎片化、信息难以复用以及决策依赖人工整理的核心问题。在日常工作中，大量信息分散在笔记、网页和聊天记录中，传统工具难以实现结构化沉淀与高效检索，导致知识利用率低、重复劳动多。  系统采用多 Agent 协同的方式构建完整闭环能力。首先由解析 Agent对输入内容进行结构化提取，生成标准化知识要点；随后由存储 Agent将内容向量化并写入向量数据库，实现长期记忆；在查询阶段，由检索 Agent基于语义相似度召回相关知识片段；再由推理 Agent（RAG）结合上下文进行综合分析并生成答案；最后通过总结 Agent对阶段性知识进行归纳，输出关键结论和行动建议。
